# 🗼 Site Paris — A Cidade das Luzes

Uma página web elegante, interativa e totalmente responsiva dedicada a explorar o charme, a gastronomia e as atrações turísticas de Paris. O projeto foi estruturado com foco em boas práticas de semântica e acessibilidade.

---

## 🚀 Funcionalidades Principais (A Cereja do Bolo)

*   **🌓 Modo Noturno Dinâmico:** Um botão flutuante inteligente que permite "apagar as luzes" do site, alternando para um modo escuro confortável para leitura noturna.
*   **⏰ Saudação Baseada no Horário:** O título principal do site se adapta dinamicamente usando JavaScript, saudando o visitante com um *"Bonjour!"* pela manhã ou um *"Bonsoir!"* durante a noite.
*   **🗺️ Interatividade nas Atrações:** Clique em qualquer ponto turístico listado para receber uma saudação e um incentivo personalizado para a sua próxima viagem.
*   **📱 Design Responsivo:** Totalmente otimizado para ser visualizado em computadores, tablets ou celulares.

---

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando as tecnologias fundamentais da web, sem a necessidade de frameworks externos:

*   **HTML5:** Estruturação semântica e acessível (uso de tags como `<article>`, `<section>`, `<figure>`, entre outras).
*   **CSS3:** Estilização moderna, uso de variáveis de transição suave e design responsivo.
*   **JavaScript (Vanilla JS):** Manipulação de DOM para os efeitos interativos e lógica de tempo do sistema.
